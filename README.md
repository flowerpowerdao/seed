# SEED token

Canister `fua74-fyaaa-aaaan-qecrq-cai`

Symbol `SEED`

Decimals `8`

Fee `0.01`


### Local development
1. Clone repository
2. Run `run git submodule init` and `git submodule update`
3. Run `mops install`
4. Start local replica and deploy:
```
npm start
```

### Deploy to production
```
npm run deploy-ic
```
